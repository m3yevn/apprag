# Apprag

Professional README auto-generator from `package.json` + `apprag.config.js`.

- **Site:** [apprag.vercel.app](https://apprag.vercel.app)

## Install

```sh
npm i -g apprag
```

## Usage

```sh
apprag --config   # create apprag.config.js
apprag            # generate README.md
```

## Develop

```sh
npm install
npm test
npm run build
```
